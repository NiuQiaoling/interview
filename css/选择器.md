### 选择器包括:
- 通配符选择器

        *{
            margin: 0;
        }

- ID选择器

        #id {
            color: red;
        }
    id名在同一个文档中是惟一的(是种规范, 可能在一些浏览器中并不会报错, 但是当使用document.getElementById('')是只会选择第一个), 因此id名不能重复


- 类选择器

        .name {
            
        }

- 元素选择器

        p {

        }

- 属性选择器



- 后代(包含)选择器

    .div .p1 {}
- 兄弟选择器

    .div + p {}

- 子选择器

    .div > p {}

- 伪类
